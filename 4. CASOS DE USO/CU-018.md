CU-018 – Visualización Web del Sistema
Información General

ID: CU-018

Nombre: Visualización Web del Sistema

Descripción breve: Permite a los usuarios acceder y navegar por la plataforma mediante un navegador web, garantizando disponibilidad y compatibilidad multiplataforma.

Módulo: Usabilidad / Escalabilidad

Prioridad: MUST

Complejidad: Media

Estado: En Revisión

Actores

Actor Principal: Todos los usuarios

Actores Secundarios: Sistema

Precondiciones

El usuario debe tener acceso a un navegador web compatible.

El sistema debe estar desplegado en un servidor accesible por la red.

Postcondiciones

El usuario puede navegar y utilizar las funcionalidades disponibles del sistema a través de la web.

Flujo Principal (Camino Feliz)

El usuario abre el navegador web.

Ingresa la URL del sistema.

El sistema valida la conexión y muestra la pantalla de inicio de sesión.

El usuario se autentica.

El sistema despliega la interfaz web según el rol del usuario.

El usuario interactúa con las funcionalidades disponibles de manera fluida y responsiva.

Flujos Alternativos

A1 – Dispositivo móvil:

El usuario accede desde un dispositivo móvil.

El sistema adapta la interfaz para ser responsiva y funcional en pantallas pequeñas.

Flujos de Excepción

E1 – Problema de conexión:

Si no hay conexión con el servidor, el sistema muestra un mensaje de error.

El usuario puede reintentar la conexión.

Reglas de Negocio

RN-35: La plataforma web debe ser compatible con los navegadores más comunes (Chrome, Edge, Firefox). (RNF-005.1)

RN-36: La disponibilidad del sistema debe mantenerse al menos un 99% del tiempo.

Requisitos No Funcionales

RNF-005.1: Desarrollado como aplicación web.

RNF-002.2: Respuesta rápida al usuario durante la navegación.

RNF-002.5: Soporte a gran volumen de usuarios simultáneos.

Interfaces de Usuario

Pantalla de inicio de sesión.

Dashboard principal según rol.

Navegación por menús y módulos del sistema.

Referencias

RNF-005.1, RNF-002.2, RNF-002.5: Escalabilidad y rendimiento web.

HU-001, HU-002, HU-003: Interacción intuitiva y accesible desde navegador web.
