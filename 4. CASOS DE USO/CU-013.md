CU-013 – Gestión de Contraseñas
Información General

ID: CU-013

Nombre: Gestión de Contraseñas

Descripción breve: Permite a los usuarios internos cambiar o recuperar sus contraseñas cumpliendo políticas de seguridad.

Módulo: Gestión de Usuarios y Roles

Prioridad: MUST

Complejidad: Media

Estado: En Revisión

Actores

Actor Principal: Usuario interno

Actores Secundarios: Sistema

Precondiciones

El usuario debe estar registrado en el sistema.

El usuario debe conocer su contraseña actual para cambio, o contar con medios de recuperación.

Postcondiciones

La contraseña del usuario queda actualizada cumpliendo las políticas de seguridad.

La acción queda registrada para auditoría. (RNF-003.4)

Flujo Principal (Camino Feliz)

El usuario accede a la opción de “Cambiar o recuperar contraseña”.

Ingresa la contraseña actual (si aplica) o el correo para recuperación.

El sistema valida la información proporcionada.

El usuario ingresa la nueva contraseña cumpliendo políticas de complejidad.

El sistema actualiza la contraseña y confirma la operación.

La acción queda registrada en el historial de auditoría.

Flujos Alternativos

A1 – Recuperación vía correo electrónico:

Si el usuario olvidó la contraseña, el sistema envía un enlace de recuperación seguro.

A2 – Contraseña temporal:

El sistema puede generar una contraseña temporal que el usuario debe cambiar al ingresar.

Flujos de Excepción

E1 – Contraseña débil:

Si la nueva contraseña no cumple los requisitos de seguridad, el sistema rechaza el cambio y muestra mensaje de error.

E2 – Intentos fallidos de recuperación:

El sistema bloquea temporalmente la opción tras varios intentos fallidos.

Reglas de Negocio

RN-25: Las contraseñas deben cumplir políticas de complejidad, longitud y caducidad.

RN-26: Todos los cambios de contraseña deben registrarse en auditoría. (RNF-003.4)

Requisitos No Funcionales

RNF-001.1: Interfaz clara y sencilla para gestión de contraseñas.

RNF-003.2: Cumplimiento de políticas de seguridad para contraseñas.

RNF-003.4: Registro de auditoría de cambios.

Interfaces de Usuario

Pantalla de cambio de contraseña.

Pantalla de recuperación vía correo electrónico.

Mensajes de validación y confirmación.

Referencias

RNF-003.2: Gestión de contraseñas y políticas de seguridad.

RNF-003.4: Registro de auditoría de acciones del usuario.
