CU-003: ACTUALIZAR INFORMACIÓN DE PARTICIPANTE
INFORMACION GENERAL

ID: CU-003

Nombre: Actualizar datos de participante

Descripción breve: Permite modificar información previamente registrada para corregir errores o mantener los datos actualizados.

Módulo: Gestión de Usuarios y Roles – Participantes

Prioridad: MUST

Complejidad: Media

Estado: Aprobado

Actores

Actor Principal: Personal administrativo

Actores Secundarios: Sistema

Precondiciones

El usuario debe estar autenticado como personal administrativo.

El participante debe existir previamente en la base de datos.

Postcondiciones

Los datos del participante quedan actualizados.

Se registra la acción en el historial de auditoría.

Flujo Principal (Camino Feliz)

El actor accede al módulo de búsqueda de participantes.

El actor localiza al participante por documento o nombre.

El actor selecciona la opción “Editar”.

El sistema muestra el formulario con los datos actuales.

El actor modifica los campos necesarios.

El sistema valida la información ingresada.

El sistema guarda los cambios.

El sistema confirma la actualización exitosa.

Flujos Alternativos

A1 – Edición parcial

El actor modifica solo algunos campos y guarda sin afectar el resto.

Flujos de Excepción

E1 – Fallo de validación

El sistema muestra los campos incorrectos y solicita corregirlos.

E2 – Cambio conflictivo

Si el nuevo número de documento coincide con uno ya existente, el sistema bloquea el cambio.

Reglas de Negocio

RN-001: No puede existir duplicidad en número de documento.

RN-004: Solo personal autorizado puede realizar cambios.

RN-006: Toda modificación debe quedar registrada para auditoría.

Requisitos No Funcionales

RNF-001.1: Interfaz sencilla para edición de datos.

RNF-001.3: Debe permitir navegación mediante teclado y ratón.

RNF-002.2: Validación y guardado deben ejecutarse con buen tiempo de respuesta.

RNF-002.4: Debe permitir edición rápida para personal experto.

RNF-003.3: Protección de datos y cambios sensibles.

RNF-003.4: Registro automático de auditoría.

RNF-005.1: Debe realizarse mediante aplicación web.

Interfaces de Usuario

Pantalla: Formulario de Edición de Participante

Elementos:

Campos personales y de contacto

Botón “Guardar cambios”

Alertas de validación

Indicador de éxito
