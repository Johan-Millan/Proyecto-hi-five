CU-005 – Gestionar Información de Instructores
Información General

ID: CU-005

Nombre: Gestionar Información de Instructores

Descripción breve: Permite consultar, actualizar o desactivar la información registrada de los instructores.

Módulo: Gestión de Usuarios y Roles

Prioridad: MUST

Complejidad: Media

Estado: En Revisión

Actores

Actor Principal: Personal Administrativo

Actores Secundarios: Sistema (validación, auditoría)

Precondiciones

El usuario debe estar autenticado y contar con permisos para gestionar instructores. (REF-03)

Debe existir al menos un instructor registrado en el sistema. (REF-01)

Postcondiciones

La información del instructor queda actualizada, consultada o desactivada según la acción. (REF-01)

Se registra la acción en el log de auditoría. (REF-03)

Flujo Principal (Camino Feliz)

El actor accede al módulo “Gestión de Instructores”. (REF-01)

El sistema muestra la lista de instructores registrados. (REF-02)

El actor selecciona un instructor para gestionar. (REF-01)

El sistema muestra las opciones: “Consultar”, “Actualizar” o “Desactivar”. (REF-02)

El actor selecciona la acción deseada. (REF-01)

Si la acción es “Actualizar”, el actor modifica los datos necesarios. (REF-01)

El sistema valida los datos ingresados. (REF-04)

El sistema guarda los cambios o ejecuta la acción elegida. (REF-01)

El sistema registra la acción en la auditoría. (REF-03)

El sistema confirma la operación realizada. (REF-01)

Flujos Alternativos
A1 – Consulta de Instructor

5a. El actor selecciona “Consultar”.

5b. El sistema muestra la información completa del instructor.
(REF-01)

A2 – Desactivación de Instructor

5c. El actor selecciona “Desactivar”.

5d. El sistema solicita confirmación.

5e. El actor confirma la desactivación.

5f. El sistema marca el instructor como inactivo.
(REF-01 + REF-03)

Flujos de Excepción
E1 – Datos inválidos al actualizar

7a. El sistema detecta errores en campos editados.

7b. Muestra los campos a corregir.

7c. El actor regresa al paso 6.
(REF-04)

E2 – Problema al guardar

8a. Ocurre un problema al guardar la actualización o desactivación.

8b. El sistema notifica el fallo al actor.
(REF-05)

Reglas de Negocio

RN-03: Registro y gestión completa de información de instructores. (REF-01)

RN-09: Todas las acciones deben quedar registradas en la auditoría. (REF-03)

RN-10: No se permite duplicar información por documento. (REF-03)

Requisitos No Funcionales

RNF-001.1: Interfaz simple e intuitiva. (REF-04)

RNF-001.3: Operable con teclado y ratón. (REF-04)

RNF-002.4: Operación eficiente para usuarios administrativos expertos. (REF-04)

RNF-003.3: Protección de datos personales de instructores. (REF-04)

RNF-003.4: Auditoría obligatoria en cada acción. (REF-04)

RNF-005.1: Funciona como aplicación web. (REF-04)

Interfaces de Usuario

Opciones: Consultar, Actualizar, Desactivar.

Tabla/listado de instructores. (REF-02)
