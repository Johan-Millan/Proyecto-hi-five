# CU-005 – Gestionar Información de Instructores

## Información General
- **ID:** CU-005
- **Nombre:** Gestionar Información de Instructores
- **Descripción breve:** Permite consultar, actualizar o desactivar la información registrada de los instructores.
- **Módulo:** Gestión de Usuarios y Roles
- **Prioridad:** MUST
- **Complejidad:** Media
- **Estado:** En Revisión
## Actores
- **Actor Principal:** Personal Administrativo
- **Actores Secundarios:** Sistema (validación, auditoría)
## Precondiciones
- El usuario debe estar autenticado y contar con permisos para gestionar instructores.
- Debe existir al menos un instructor registrado en el sistema.
## Postcondiciones
- La información del instructor queda actualizada, consultada o desactivada según la acción.
- Se registra la acción en el log de auditoría.
## Flujo Principal (Camino Feliz)
1. El actor accede al módulo “Gestión de Instructores”.
2. El sistema muestra la lista de instructores registrados.
3. El actor selecciona un instructor para gestionar.
4. El sistema muestra las opciones: “Consultar”, “Actualizar” o “Desactivar”.
5. El actor selecciona la acción deseada.
6. Si la acción es “Actualizar”, el actor modifica los datos necesarios.
7. El sistema valida los datos ingresados.
8. El sistema guarda los cambios o ejecuta la acción elegida. 
9. El sistema registra la acción en la auditoría. 
10. El sistema confirma la operación realizada. 
## Flujos Alternativos
### A1 – Consulta de Instructor
- El actor selecciona “Consultar”.
- El sistema muestra la información completa del instructor.
### A2 – Desactivación de Instructor
- El actor selecciona “Desactivar”.
- El sistema solicita confirmación.
- El actor confirma la desactivación.
- El sistema marca el instructor como inactivo.
## Flujos de Excepción
### E1 – Datos inválidos al actualizar
- El sistema detecta errores en campos editados.
- Muestra los campos a corregir.
- El actor regresa al paso 6.
### E2 – Problema al guardar
- Ocurre un problema al guardar la actualización o desactivación.
- El sistema notifica el fallo al actor.
## Reglas de Negocio
- RN-03: Registro y gestión completa de información de instructores. 
- RN-09: Todas las acciones deben quedar registradas en la auditoría. 
- RN-10: No se permite duplicar información por documento.
