CU-015 – Respaldo de Datos
Información General

ID: CU-015

Nombre: Respaldo de Datos

Descripción breve: Ejecuta respaldos automáticos de la información del sistema para garantizar la continuidad y seguridad de los datos.

Módulo: Gestión de Usuarios y Roles / Fiabilidad

Prioridad: MUST

Complejidad: Media

Estado: En Revisión

Actores

Actor Principal: Sistema

Actores Secundarios: Administrador del sistema

Precondiciones

El sistema debe estar operativo.

Disponibilidad de almacenamiento local y en la nube.

Postcondiciones

Se generan respaldos completos o incrementales de la base de datos.

Los respaldos quedan almacenados de forma segura y verificable.

Flujo Principal (Camino Feliz)

El sistema inicia la rutina de respaldo según la programación configurada.

Se realiza una copia de seguridad de la base de datos y archivos del sistema.

El sistema almacena los datos en almacenamiento local y en la nube.

El sistema verifica la integridad de los respaldos.

Se notifica al administrador que el respaldo se realizó correctamente.

Flujos Alternativos

A1 – Respaldo manual:

El administrador solicita un respaldo inmediato.

El sistema ejecuta los pasos de respaldo como en el flujo principal.

Flujos de Excepción

E1 – Fallo de almacenamiento:

Si falla la escritura en el almacenamiento local o en la nube, el sistema genera alerta al administrador.

Reintenta el respaldo hasta que se complete correctamente.

Reglas de Negocio

RN-29: Todos los respaldos deben incluir la base de datos completa y archivos críticos del sistema.

RN-30: Los respaldos deben realizarse automáticamente según la programación establecida. (RNF-004.1, RNF-004.3)

Requisitos No Funcionales

RNF-004.1: Estrategia de respaldo robusta.

RNF-004.3: Respaldo combinado: local y nube.

RNF-002.2: La operación debe ejecutarse en tiempo de respuesta aceptable.

Interfaces de Usuario

Panel de administración de respaldos.

Alertas y notificaciones de estado de respaldo.

Referencias

RNF-004.1, RNF-004.3: Estrategia de respaldo robusta y combinada.

RF-002.3: Repositorio central para auditoría y continuidad del sistema.
