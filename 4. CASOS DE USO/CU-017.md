CU-017 – Ajustes de Accesibilidad
Información General

ID: CU-017

Nombre: Ajustes de Accesibilidad

Descripción breve: Permite al usuario modificar configuraciones de accesibilidad para facilitar la interacción con el sistema, incluyendo tamaño de letra, navegación por teclado e iconografía accesible.

Módulo: Usabilidad

Prioridad: SHOULD

Complejidad: Media

Estado: En Revisión

Actores

Actor Principal: Usuario del sistema

Actores Secundarios: Sistema

Precondiciones

El usuario debe estar autenticado en el sistema.

El módulo de ajustes de accesibilidad debe estar habilitado.

Postcondiciones

Los ajustes seleccionados se aplican y permanecen guardados para futuras sesiones.

Flujo Principal (Camino Feliz)

El usuario accede al módulo de ajustes de accesibilidad.

Selecciona las opciones deseadas: tamaño de letra, navegación con teclado, contraste, iconografía, etc.

El sistema aplica los cambios de manera inmediata en la interfaz.

El sistema guarda las preferencias del usuario.

El usuario confirma que los cambios mejoran la accesibilidad y la interacción.

Flujos Alternativos

A1 – Restablecer ajustes predeterminados:

El usuario decide restaurar la configuración original del sistema.

El sistema restablece los valores por defecto y confirma al usuario.

Flujos de Excepción

E1 – Error al aplicar ajustes:

Si ocurre un error durante la aplicación de los cambios, el sistema notifica al usuario.

Se permite reintentar la configuración o restaurar los valores predeterminados.

Reglas de Negocio

RN-33: Todos los cambios de accesibilidad deben ser reversibles por el usuario. (RNF-001.2, RNF-001.4)

RN-34: Las configuraciones aplicadas no deben afectar la seguridad ni la integridad de los datos del sistema.

Requisitos No Funcionales

RNF-001.2: Sistema accesible para personas con discapacidades, priorizando iconografía y señales visuales.

RNF-001.3: Operable con teclado y ratón.

RNF-001.4: Permitir cambiar el tamaño de la fuente.

Interfaces de Usuario

Panel de ajustes de accesibilidad.

Vista previa en tiempo real de cambios de interfaz.

Botón de restauración de configuración predeterminada.

Referencias

RNF-001.2, RNF-001.3, RNF-001.4: Usabilidad y accesibilidad de la interfaz.

HU-001, HU-002: Mantener interacción intuitiva y funcionalidad para todos los usuarios.
